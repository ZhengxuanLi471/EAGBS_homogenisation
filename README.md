# EAGBS Microstructural Heterogeneity — Code Repository

> Repository preparation (README, workflow documentation, and bug fixes) assisted by [Claude](https://claude.ai) (Anthropic).

## Dependencies

| Software | Purpose |
|----------|---------|
| [Neper](https://neper.info) ≥ 4.x | Voronoi tessellation generation |
| [NGSolve](https://ngsolve.org) + Netgen | Finite-element solver |
| Python ≥ 3.8 | All scripts |
| numpy, scipy, pandas, matplotlib | Python packages |

Install Python packages: `pip install numpy scipy pandas matplotlib`

## Repository Structure

```
eagbscodes/
├── geometry_generation/     # Step 1: tessellation generation & pre-processing
│   ├── generate_jobs.py     # Prints bash commands to run Neper
│   ├── tile4_to_square.py   # Clips periodic Voronoi to unit square
│   └── read.py              # Converts PLY files → tessellation_output.json
├── grain_size/              # Step 2: grain-size heterogeneity sweep
│   ├── main.py              # NGSolve mixed FE solver
│   ├── meshes.py            # Mesh construction & grain-boundary labelling
│   ├── physics.py           # Constitutive utilities
│   ├── real_im_energy.py    # Driver: sweeps ω for each tessellation seed
│   └── real_im_hex_energy.py# Hexagonal benchmark (validation)
├── gbviscosity/             # Step 3: GB-viscosity distribution sweep
│   ├── main.py              # NGSolve solver with per-boundary viscosity
│   ├── meshes.py
│   ├── physics.py
│   ├── run_eta_sweep.py     # Driver: sweeps σ_η and ω for one seed
│   ├── compute_gb_lengths.py# Post-processing: boundary length per viscosity bin
│   └── plot_eta_sweep.py    # Visualisation of viscosity-sweep results
├── fit_Priestley/           # Step 4: seismic model fitting
│   ├── fitting_self_P.py    # Fits reduced-order EAGBS model to surface-wave data
│   └── keith.dat            # Priestley (2024) surface-wave velocities
└── plotting/                # Step 5: publication figures
    ├── sigmas/              # Grain-size sweep output — generated by Step 2 (not tracked)
    ├── plt_Cxyxy_pub.py     # Figure: G'(ω), G''(ω) vs σ_a (grain-size effect)
    └── visualize_mesh_sigma_comparison.py  # Figure: tessellation comparison
```

Large generated files (`plotting/sigmas/`, `tessellation_output.json`, `eta_sweep_results/`) are excluded from this repository. Follow the workflow below to reproduce them.

## Workflow

### Step 1 — Generate Tessellations (requires Neper)

```bash
cd geometry_generation

# Generate bash job script and run it (creates sigma_0.05/ … sigma_0.50/)
python generate_jobs.py > run_neper.sh
bash run_neper.sh

# Inside each sigma_* directory, convert PLY files to the JSON format
# required by the FE solvers:
for d in sigma_*/; do
    cp read.py "$d"
    (cd "$d" && python read.py)
done
```

Each `sigma_*/tessellation_output.json` maps `"seeds_1"…"seeds_100"` to `[pts, regions]` and is the required input for Steps 2 and 3.

---

### Step 2 — Grain-Size Heterogeneity Sweep (requires NGSolve, HPC recommended)

Run for each σ_a level (0.05, 0.10, …, 0.50). Example for σ_a = 0.05:

```bash
cd grain_size

# Make the tessellation JSON available in the working directory:
cp ../geometry_generation/sigma_0.05/tessellation_output.json .

# Frequency sweep over all 100 seeds (outputs one CSV per seed):
python real_im_energy.py

# Hexagonal benchmark (single reference geometry):
python real_im_hex_energy.py

# Move outputs to the plotting data directory:
mkdir -p ../plotting/sigmas/sigma_0.05
mv *.csv ../plotting/sigmas/sigma_0.05/
```

Repeat, substituting the appropriate `sigma_*` directory, for each of the 10 σ_a levels.

---

### Step 3 — GB-Viscosity Distribution Sweep (requires NGSolve, HPC recommended)

Each call to `run_eta_sweep.py` processes one tessellation seed across all σ_η values.
Run for as many seeds as desired (the paper uses seeds 1–100 from σ_a = 0.05).

```bash
cd gbviscosity

# Example: sweep seed 1 under shear loading
python run_eta_sweep.py \
    --tess-json ../geometry_generation/sigma_0.05/tessellation_output.json \
    --seed seeds_1 \
    --output-dir eta_sweep_results \
    --load shear

# Repeat for seeds_2 … seeds_100; results accumulate in eta_sweep_results/

# Visualise once data collection is complete:
python plot_eta_sweep.py --results-dir eta_sweep_results --output-dir figures
```

---

### Step 4 — Seismic Model Fitting (standalone, no HPC needed)

```bash
cd fit_Priestley
python fitting_self_P.py
```

Reads `keith.dat` (included) and outputs best-fit parameters and figures.

---

### Step 5 — Publication Figures

```bash
cd plotting

# Grain-size effect figures (reads from plotting/sigmas/ generated in Step 2):
python plt_Cxyxy_pub.py

# Tessellation comparison figure (reads tessellation_output.json from Step 1):
python visualize_mesh_sigma_comparison.py
```
